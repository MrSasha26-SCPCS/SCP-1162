# SCP-1162
SCP: Classified Site plugin

Данный плагин добавляет нового неиграбельного SCP - "Дыра в стене". Он появляется в комнате на этаже с камерой содержания SCP-173. С помощью этого объекта можно обменять почти любой предмет на другой (как при прокачке в SCP-914 на 1:1) раз в 5 секунд.

**Установка:**
1) `.dll`/`.lua` файл вставить туда, где находятся обычные плагины (в папку `Plugins`);
2) Содержимое архива распаковать в папку `Plugins/client`.

**Если возникают ошибки:** попробуйте не распаковывать архив прямо на сервере, а сначала создайте в папке с плагинами папку `client`, а в ней папки `Android`, `Windows`, `Linux` и `OSX`, и в них поместите бандл файлы (помните, что их путать нельзя, каждый в своей папке!).
